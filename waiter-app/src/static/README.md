# Static 资源目录

本目录用于存放静态资源文件，包括：

## TabBar 图标

将以下图标文件放入本目录（建议尺寸 81x81 或 162x162 PNG 格式）：

| 文件名 | 说明 |
|--------|------|
| `table.png` | 桌位 - 未选中状态 |
| `table-active.png` | 桌位 - 选中状态 |
| `order.png` | 订单 - 未选中状态 |
| `order-active.png` | 订单 - 选中状态 |
| `call.png` | 呼叫 - 未选中状态 |
| `call-active.png` | 呼叫 - 选中状态 |
| `mine.png` | 我的 - 未选中状态 |
| `mine-active.png` | 我的 - 选中状态 |

## 占位图

| 文件名 | 说明 |
|--------|------|
| `default-food.png` | 默认菜品图片占位图（建议尺寸 400x400） |

> **注意**：图标颜色建议：
> - 未选中：灰色 (#969799)
> - 选中：主色调蓝绿色渐变 (#1989fa / #07c160)
