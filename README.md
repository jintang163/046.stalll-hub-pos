# 大排档餐饮系统 (Stall Hub POS)

全流程数字化餐饮管理系统，支持多端协同、离线收银、小程序点餐。

## 技术架构

```
┌─────────────────────────────────────────────────────────────────┐
│                             客户端                              │
├─────────────────┬─────────────────┬─────────────────────────────┤
│   管理台        │   收银端        │      小程序                 │
│ Vue3 + Element  │ Electron + Vue3 │  Taro + NutUI (React)       │
│   Plus          │  Pinia + SQLite │                             │
└────────┬────────┴────────┬────────┴──────────────┬──────────────┘
         │                 │                       │
         │                 │                       │
┌────────▼─────────────────▼───────────────────────▼──────────────┐
│                          API Gateway                            │
│                         Go + Gin                                │
├─────────────────────────────────────────────────────────────────┤
│                          业务服务层                              │
│  商品服务 | 订单服务 | 会员服务 | 支付服务 | 库存服务 | 报表服务  │
├─────────────────────────────────────────────────────────────────┤
│                          数据层                                 │
│  GORM + MySQL  |  Redis  |  NSQ  |  MinIO                      │
└─────────────────────────────────────────────────────────────────┘
```

## 核心功能

### 1. 商品管理
- 多SKU管理（大/中/小杯独立价格、库存、图片）
- 多属性管理（辣度等独立库存）
- 批量改价、商品复制、上下架
- 库存预警（阈值可配置）

### 2. 收银端
- SQLite 离线缓存商品数据
- NSQ 实时订阅商品变更
- 断网续传、订单本地队列
- 全量/增量同步进度展示

### 3. 小程序
- 门店选择、商品分类展示
- SKU/属性选择、购物车
- 微信支付、订单状态追踪
- 退款/售后申请

### 4. 高级功能
- 门店管理、打印机分单
- 会员积分、优惠券系统
- 营业报表、数据分析
